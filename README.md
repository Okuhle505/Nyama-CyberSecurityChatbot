**CyberSecurity Awareness Chatbot**

**Project Overview**

This project is a Cybersecurity Awareness Chatbot develeped in C#. The Chatbot helps educate users about cybersecurity topics such as password safety,
phishing scams, online privacy, and safe browsing habits.

The project consists of:

Part 1: Console-based chatbot

Part2: GUI-based chatbot using Windows Forms

**Features**

**Part 1**
- Console Chatbot interaction
- Voice Greeting using WAV audio
- User input and responses
- Input Validation
- Cybersecurity tips

**Part2**
- Windows fors GUI
- Keyword recognition
- Conversation flow
- Improved user interface
- Random chatbot responses

**Cybersecurity Topics Covered**
- Password safety
- Phishing scams
- Privacy protection
- Online scam awareness

**Technologies Used**
- C#
- .NET Framework 4.8
- Windows Forms
- GitHub

**How to Run the Application**
1. Open the solution in Visual Studio 2022.
2. Build the solution
3. Run either:
   - CyberSecurityChatbot.Console
   - CyberSecurityChatbot.GUI
